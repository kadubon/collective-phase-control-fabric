# Dependencies

Core dependencies are `jsonschema>=4.26,<5` for Draft 2020-12 validation,
`networkx>=3.6.1,<4` for bounded finite graph routines, and `cryptography>=49,<50` for mandatory
pinned Ed25519 verification. They use permissive licenses. Build, test, type, lint, security,
audit, and packaging tools are development-only extras. CPCF vendors no dependency source.
