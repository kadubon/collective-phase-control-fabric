# Analogy Boundary

`formation_seed` is a software dependency object, not a physical critical nucleus.
`regeneration_deadlock` is a bounded dependency diagnostic, not general chemical-siphon enumeration.
P/T-semiflow analogues are exact coordinate identities, not chemical conservation laws. Resource
potentials are user-declared accounting checks, not thermodynamic potentials. Rate intervals are
external observations, not a kinetic simulation. The independent support core is not a physical
k-core phase, and an intervention cover is not general controllability. `phase_label` is user text
and cannot promote status. `structural_progress` means a receipt-backed ordering improvement only.
