# Operational Organization Profile

CPCF reports thirteen independent dimensions. Each is `satisfied`, `violated`, `unknown`, or
`unknown_due_to_budget`.

| Dimension | Operational question |
|---|---|
| provenance integrity | Are required live records reconstructable from admitted sources? |
| trust quorum | Do high-impact decisions satisfy role separation? |
| temporal integrity | Is lifecycle status established by trusted time? |
| structural reachability | Are targets reachable in the typed hypergraph? |
| causal formation | Is there a strictly prior formation sequence? |
| dimensional consistency | Are quantities, actions, and time dimensions compatible? |
| exact self-maintenance | Does a target-bound closed organization have exact maintenance flux? |
| finite-horizon resource persistence | Do all trajectory prefixes preserve floors? |
| target-bound generative catalysis | Does admitted food/catalysis form targets without circular credit? |
| verification capacity | Can required evidence routes be serviced under declared bounds? |
| effective independence | Does admitted exposure evidence support enough independent domains? |
| coordination protocol integrity | Is the signed coordination state machine complete and valid? |
| perturbation robustness | Do required reduced-snapshot audits preserve all required dimensions? |

`operational_organization_compatible` is true only when every contract-required dimension is
satisfied in one immutable snapshot. The dimensions are not collapsed into a scalar intelligence
score. A favorable structural profile is separate from externally registered acceleration evidence.
