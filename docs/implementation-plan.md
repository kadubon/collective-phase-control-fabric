1. Establish Apache-2.0 packaging, versioned JSON Schemas, canonical JSON, and a local
   content-addressed store.
2. Implement a finite monotone AND/OR directed hypergraph and fail-closed feasible and verified
   closures.
3. Recompute supplied productive and maintenance witnesses with exact rational arithmetic.
4. Detect false-positive cycles, duplicate mass, stale support, proxy-only support, resource
   sinks, authority loops, and regeneration deadlocks.
5. Compute bounded formation seeds, coordinate-wise barriers, structural sensitivity, verification
   load, and duration-gated critical paths.
6. Plan one finite action through ordered hard filters, conditional postcondition replay, Pareto
   selection, and stagnation exclusion.
7. Provide deterministic fixtures, a fake check-only adapter, transition records, portable bundles,
   and the complete first-time-agent CLI path.
8. Pass the M0 tests and quality gates before enabling verified CCR/PIC read-only projections.
9. Add fixture-only ALT, PCS, FOST/VEK, CMGL, and external-certificate handoffs without implying
   downstream acceptance.
10. Verify packaging, schemas, fixtures, security constraints, installed-wheel behavior, and write
    the release-readiness report.

The implementation does not search for productive witnesses, enumerate general hyperpaths or
minimal siphons, execute external effects, invoke an LLM, run experiments, or publish artifacts.
